# RB-03 Copilot-First FinOps Operating System

## Visual and report deliverables

### Daily visual deliverables

- anomaly leaderboard by impact,
- top driver breakdown view,
- high-risk unresolved queue.

### Weekly report deliverables

- trend comparison (WoW),
- backlog aging and closure chart,
- expected vs realized impact summary,
- forecast drift note with confidence changes.

### Monthly report deliverables

- actual vs forecast vs budget narrative,
- top recurring drivers with root-cause status,
- commitment candidate options (upside/downside),
- governance and escalation log.

## Copilot prompt set for report generation

1. "Generate an executive-ready weekly cost summary with top risks and decisions needed."
2. "Generate an engineering-manager report with owner-ready actions and rollback notes."
3. "Generate an FP&A forecast variance report with confidence bands."
4. "Generate a procurement commitment options report with downside-risk commentary."
